# Private_Poll_API
API for private poll<br>

<b>Account status:</b><br>
0: Logged out<br>
1: Logged in<br>
2: Suspended<br>
<br>
<b>Poll status:</b><br>
0: Pending<br>
1: Active<br>
2: Completed<br>
3: Cancelled<br>
<br>
<br>
